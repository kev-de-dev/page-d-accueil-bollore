Mes Commandes Utiles

Commandes système utiles en ligne de commande.

  • NCPA.CPL
    Ouvre les propriétés de connexion réseau, permettant la configuration des adaptateurs et des paramètres réseau.
  • compmgmt.msc
    Ouvre la console de gestion de l'ordinateur pour diverses tâches d'administration.
  • CERTMGR.MSC
    Accède à la gestion des certificats pour les utilisateurs et les ordinateurs.
  • services.msc
    Ouvre la console des services pour la gestion des services système.
  • devmgmt.msc
    Lance le Gestionnaire de périphériques pour la gestion des périphériques matériels.
  • MMC.EXE
    ouvre l'Utilitaire de configuration système pour la configuration du démarrage.
  • msconfig
    Ouvre l'Utilitaire de configuration système pour la configuration du démarrage.
